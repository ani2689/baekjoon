# 문제
정수 A, B 가 주어진다. 세로 길이가 A cm, 가로 길이가 B cm 인 아래와 같은 직사각형의 넓이를 cm^2 단위로 구하시오.
![사각형 넓이 구하기](사각형 넓이 구하기.png)

# 입력
표준 입력에 다음과 같은 형태로 입력이 주어진다.
```
A
B
```

# 출력
세로 길이가 A cm, 가로 길이가 B cm인 직사각형의 넓이를 cm^2 단위로 구하고, 단위 (cm^2)를 생략하여 출력한다.

# 예제
### 예제 1
#### 입력 
```
2
3
```
#### 출력
```
6
```
### 예제 2
#### 입력 
```
100
1
```
#### 출력
```
100
```
### 예제 3
#### 입력 
```
4
4
```
#### 출력
```
16
```
